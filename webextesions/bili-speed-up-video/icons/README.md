# 图标文件夹

请在此文件夹中放置以下尺寸的PNG图标文件：

- `icon16.png` - 16x16 像素
- `icon32.png` - 32x32 像素  
- `icon48.png` - 48x48 像素
- `icon128.png` - 128x128 像素

## 图标设计建议

- 使用简洁的设计，在小尺寸下也能清晰识别
- 建议使用播放相关的图标，如播放按钮、速度表等
- 保持一致的设计风格和颜色方案
- 确保在浅色和深色背景下都有良好的对比度

## 临时解决方案

如果暂时没有图标文件，可以：
1. 使用在线图标生成器创建简单图标
2. 从免费图标库下载合适的图标
3. 使用Chrome默认图标（插件仍可正常工作）
